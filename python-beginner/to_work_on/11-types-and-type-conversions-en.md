# Types and type conversions

## Types

Determining the type of a variable via `type`:

```py
a = 4 / 2

type(a)
```

## Type conversions

Objects may be converted to other types via `int()`, `float()`, `str()`, `bool()`, ...

```py
pi = 3.1415
pi_int = int(pi)
message = "Pi is approximately " + str(pi_int)
```
